# 代码相关提示词模板

这个文件包含常用的代码相关提示词模板，可以在不同插件中复用。

[粘贴代码]

```

```

## 代码生成提示词

### REST API 生成

```
请为 [实体名称] 生成一个完整的 RESTful API，包括：
- GET /api/[实体]（获取列表）
- GET /api/[实体]/:id（获取单个）
- POST /api/[实体]（创建）
- PUT /api/[实体]/:id（更新）
- DELETE /api/[实体]/:id（删除）

要求：
- 使用 [框架名称]
- 包含错误处理
- 添加输入验证
- 遵循 REST 最佳实践

字段：
[列出字段及类型]
```

### 数据模型生成

```
请为 [实体名称] 创建数据模型，包括：
- 字段定义
- 数据验证
- 序列化方法
- 关系定义（如果有）

字段：
- [字段名]: [类型] - [描述]
- ...

使用 [语言/ORM]。
```

### 单元测试生成

```
请为以下函数/类生成完整的单元测试：

测试要求：

- 测试异常情况
- 测试覆盖率 > 80%

### UI 组件生成

```
请创建一个 [组件名称] 组件，功能：
[描述组件功能]

要求：
- 使用 [框架]（React/Vue/Angular）

## 使用说明

### 如何使用这些模板

1. **选择合适的模板**：根据你的需求选择对应的模板
2. **填充占位符**：将 `[...]` 部分替换为实际内容
3. **调整细节**：根据具体情况调整模板内容
4. **提交给 Claude**：使用填充好的提示词与 Claude 交互

### 自定义模板

你可以基于这些模板创建自己的版本：
- 添加项目特定的要求
- 调整详细程度
- 组合多个模板
- 创建领域特定的模板

### 最佳实践

1. **具体明确**：提供足够的上下文和细节
2. **结构清晰**：使用列表和分段组织信息
3. **示例充分**：提供具体的代码示例
4. **目标明确**：清楚说明期望的输出
