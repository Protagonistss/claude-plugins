# Command: /todo

快速创建和管理待办事项的命令。

## 描述

这个命令帮助你快速添加、列出和管理待办事项，无需离开当前工作流程。

## 用法

```
/todo <action> [arguments]
```

### 可用操作

- `add <task>` - 添加新的待办事项
- `list` - 列出所有待办事项
- `done <id>` - 标记任务为已完成
- `remove <id>` - 删除任务

## 示例

### 添加任务

```
/todo add 完成项目文档
/todo add 审查代码 PR#123
```

### 列出任务

```
/todo list
```

输出示例：
```
待办事项：
1. [ ] 完成项目文档
2. [ ] 审查代码 PR#123
3. [x] 修复 Bug #456
```

### 完成任务

```
/todo done 1
```

### 删除任务

```
/todo remove 3
```

## 特性

- 🎯 快速添加任务
- 📋 清晰的任务列表
- ✅ 标记完成状态
- 🗑️ 删除不需要的任务

## 注意事项

- 任务 ID 从 1 开始
- 任务存储在本地会话中
- 重启后任务列表会清空（未来可能支持持久化）

## 相关命令

- `/focus` - 进入专注模式
- `/timer` - 设置计时器
