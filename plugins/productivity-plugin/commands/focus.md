# Command: /focus

启动专注模式，帮助你集中精力完成重要任务。

## 描述

启用专注模式时，Claude 会：
- 提供简洁、直接的回答
- 减少不必要的解释
- 专注于任务执行
- 避免发散性讨论

## 用法

```
/focus [on|off] [task]
```

### 参数

- `on` - 开启专注模式（默认）
- `off` - 关闭专注模式
- `task` - （可选）指定要专注的任务描述

## 示例

### 开启专注模式

```
/focus on
```

### 专注于特定任务

```
/focus 完成用户认证功能
```

Claude 会回复：
```
✓ 已进入专注模式
📌 当前任务：完成用户认证功能

我将专注于帮助你完成此任务。有什么需要开始的吗？
```

### 关闭专注模式

```
/focus off
```

## 专注模式特点

- ⚡ 快速响应
- 🎯 聚焦目标
- 💡 减少冗余
- ✅ 高效执行

## 最佳实践

1. **明确任务**：在开启专注模式时指定具体任务
2. **时间限制**：建议配合番茄工作法使用
3. **避免切换**：在专注期间避免频繁切换任务
4. **任务完成后关闭**：完成后记得关闭专注模式
